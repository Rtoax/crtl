The CRTL documentation¶

This is the top level of the CRTL’s documentation tree. CRTL documentation, like the CRTL itself, is very much a work in progress; that is especially true as we work to integrate our many scattered documents into a coherent whole. Please note that improvements to the documentation are welcome; join the CRTL-doc list at github.com if you want to help out.
